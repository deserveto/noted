## Report Language And Style

- Language: Indonesian
- Tone: polished product report with academic structure
- Format: DOCX only
- Visual style: clean, professional, readable, suitable for university submission
- Use tables for team roles, package list, feature list, and verification results
- Use screenshot placeholders if final screenshots are not available
